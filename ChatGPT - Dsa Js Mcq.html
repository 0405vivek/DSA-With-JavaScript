Time Complexity & Sorting Algorithms

  1. What is the time complexity of searching an element in an unsorted array using linear search?

    • A) O(1)

    • B) O(log N)

    • C) O(N)

    • D) O(N^2)

    • Answer: C) O(N)

  2. Which sorting algorithm has the worst-case time complexity of O(N^2)?

    • A) Merge Sort

    • B) Quick Sort

    • C) Bubble Sort

    • D) Heap Sort

    • Answer: C) Bubble Sort

  3. What is the best-case time complexity of QuickSort?

    • A) O(N)

    • B) O(N log N)

    • C) O(N^2)

    • D) O(log N)

    • Answer: B) O(N log N)

  4. Which sorting algorithm is best for nearly sorted arrays?

    • A) Quick Sort

    • B) Heap Sort

    • C) Bubble Sort

    • D) Insertion Sort

    • Answer: D) Insertion Sort

  5. What is the worst-case time complexity of QuickSort?

    • A) O(N)

    • B) O(N log N)

    • C) O(N^2)

    • D) O(log N)

    • Answer: C) O(N^2)

Stack & Queue

  1. What data structure uses LIFO (Last In, First Out)?

    • A) Queue

    • B) Stack

    • C) Linked List

    • D) Graph

    • Answer: B) Stack

  2. Which data structure is used in Breadth-First Search (BFS)?

    • A) Stack

    • B) Queue

    • C) Priority Queue

    • D) Set

    • Answer: B) Queue

  3. How can you implement a queue using two stacks?

    • A) Using push and pop operations

    • B) Using only pop operations

    • C) Using shift and unshift

    • D) Using priority queues

    • Answer: A) Using push and pop operations

  4. Which operation is used to remove an element from the top of a stack?

    • A) push()

    • B) pop()

    • C) shift()

    • D) unshift()

    • Answer: B) pop()

  5. How do you check if a stack is empty?

    • A) stack.length == 0

    • B) stack.peek() == null

    • C) stack.size() == 0

    • D) stack.isEmpty()

    • Answer: A) stack.length == 0

    • A) O(1)

    • B) O(N)

    • C) O(log N)

    • D) O(N²)

    • Answer: B) O(N)

  2. How do you remove the last node of a singly linked list?

    • A) By setting the second last node’s next pointer to null

    • B) By deleting the entire list

    • C) By shifting all elements

Hashing & Sets

    • B) O(N)

    • C) O(log N)

    • D) O(N log N)

    • Answer: A) O(1)

  2. What is a Hash Collision?

    • A) When two different keys map to the same value

    • B) When two different values have the same key

    • C) When a hash function runs indefinitely

    • D) When a HashMap is empty

  3. What is the best way to handle hash collisions?

Graph Algorithms

  1. What is the time complexity of DFS (Depth-First Search) in an adjacency list?

    • A) O(V + E)

    • B) O(V²)

    • C) O(log V)

    • D) O(V log V)

    • Answer: A) O(V + E)

  2. Which data structure is used for Kruskal’s Algorithm?

    • A) Heap

    • B) Disjoint Set

    • C) Stack

    • D) Queue

    • Answer: B) Disjoint Set

  3. Which graph traversal algorithm finds the shortest path in an unweighted graph?

    • A) Depth-First Search (DFS)

    • B) Breadth-First Search (BFS)

    • C) Dijkstra’s Algorithm

    • D) Bellman-Ford Algorithm

    • Answer: B) Breadth-First Search (BFS)
